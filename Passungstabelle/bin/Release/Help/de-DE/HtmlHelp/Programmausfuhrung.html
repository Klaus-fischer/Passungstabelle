Programmausführung

Unter dem Reiter Passungstabelle im SolidWorks Commandmanager stehen 3 Symbole zur Verfügung

Das Programm wird durch einen Klick auf das entsprechende Symbol in der Werkzeugleiste "Passungstabelle" gestartet


Wenn im Setup, die Option Evengesteuert angewählt wurde,

wird das Makro bei jedem Neuaufbau angestossen bzw. auch durch die Tastenkombination "Strg+Q"


Der Setup-Dialog wird über das Symbol gestartet

Die Hilfe wird über gestartet.

Die Hilfe-Datei liegt in zwei Formaten vor, als CHM-Datei und als HTML-Hilfe.


Hinweise

Passungstabelle

die Passungstabelle erscheint unter dem Feature Blatt und hat den Namen "Passungstabelle" + "-" + "Blattname"

Kopieren von Blättern mit Passungstabelle

wird ein bestehendes Blatt mit einer Passungstabelle über Blatt->Kopieren und  Blatt->Einfügen eingefügt,

dann ändert sich der Name der Passungstabelle auf "Allgemeine Tabelle" + "fortlaufende Nummer"

Bitte diese Tabelle löschen.

Wird diese Tabelle nicht gelöscht, dann befinden sich, nach dem das Makro gestartet wurde, zwei Tabellen unter Blatt

Formatfindung

bevor die Tabelle eingefügt wird, wird die Formatvorlage des aktuellen Zeichenblattes ermittelt.

Danach wird in den Setup-Einstellungen nach diesem Format gesucht und die Tabelleneinstellungen dieses Formats angewendet.


Wird der Formatname nicht gefunden wird in den Setup-Einstellungen nach einem Format mit den gleichen Abmessungen

des aktuellen Zeichenblattes gesucht.

Wird ein Format mit den selben Abmessungen gefunden, werden die Tabelleneinstellungen dieses Formats angewendet.


Es wird, beginnend beim ersten Format in der Formatliste nach Formaten gesucht.

Gibt es mehrere Formate mit den selben Abmessungen wird das erste gefundene Format genommen


Wird auch kein Format mit gleichen Abmessungen gefunden, werden die Tabelleneinstellungen des ersten,

in den Einstellungen definierten Formats, angewendet.
