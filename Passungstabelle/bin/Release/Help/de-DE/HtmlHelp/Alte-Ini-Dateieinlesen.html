Alte-Ini-Datei einlesen

mit diesem Funktion kann eine alte Ini-Datei eingelesen und als Setup-Datei im XML Format gespeichert werden.

  • die alte Ini-Datei muss dafür in das Installationverzeichnis des Add Ins kopiert werden
  • wenn diese Funktion genutzt wird, wird eine Excel Installation vorausgesetzt
  • nach dem Import, müssen die Übersetzungen unter Sprache vervollständigt werden und die Spaltenbreiten unter Tabellenspalten
