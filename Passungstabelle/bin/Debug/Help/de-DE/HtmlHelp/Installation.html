Installation


Die Installation immer mit Rechts-Klick auf die Setup.Exe und der Option "Als Administrator ausführen" installieren

Sollte das Setup versehentlich nicht als Administrator ausgeführt werden und auf ein lokales Verzeichnis installiert,

auf das kein Schreibrecht besteht, dann werden die Installationsdateien wahrscheinlich im Verzeichnis "C:\" landen.

Falls das passiert einfach über die Systemsteuerung deinstallieren und neu installieren



Unter "Ordner" das Installationsverzeichnis für das Add-In angeben

vorzugsweise ist das ein lokales Verzeichnis

Unter "Setup Ordner" ein Verzeichnis für die Setup.XML Datei angeben
  • das kann ein Netzwerkverzeichnis sein, damit z.B. alle Benutzer die gleichen Einstellungen verwenden
  • das Verzeichnis und/oder die Datei kann auch für bestimmte Benutzer schreibgeschützt sein
    damit ein versehentliches ändern der Einstellungen verhindert werden kann
  • Eine bereits vorhandene Setup.XML Datei wird, bei einer neuerlichen Installation, nicht überschrieben
  • Die Datei Setup.XML kann auch in das Installationsverzeichnis gespeichert werden
    allerdings muss dann darauf geachtet werden, dass auf dieses Verzeichnis Schreibzugriff besteht


Typische Installation für eine Mehr-Benutzer-Umgebung
  • Add-Inn wird jeweils lokal installiert
  • Setup.XML wird in ein Netzwerkverzeichnis installiert und bei der Installation auf den einzelnen Rechner
    wird immer auf das gleiche Verzeichnis für die Setup.XML verwiesen


Die Installation des Setup muss in ein Verzeichnis erfolgen, auf das Sie Schreibzugriff haben, damit die Setup-Datei gespeichert werden kann


wird die Installation in ein Netzwerkverzeichnis durchgeführt, dann wird wahrscheinlich die CHM-Hilfe-Datei nicht korrekt angezeigt.

Das ist kein Programmfehler sondern, eine Einschränkung von Windows.

Das ist auch der Grund, warum die von SolidWorks aus gestartete Hilfe die HTML-Hilfe ist.



nach der Installation kann das Makro über das SolidWorks Menü

Extras->Zusatzanwendungen

aktiviert oder deaktiviert werden
