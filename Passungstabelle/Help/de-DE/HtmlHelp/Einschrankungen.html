Einschränkungen

Skizzen

wenn aktiviert, werden Passungen von Skizzen berücksichtigt

wenn deaktiviert, werden keine Passungen von Skizzen berücksichtigt

Teile

wenn aktiviert, werden Passungen von Teilen berücksichtigt

wenn deaktiviert, werden keine Passungen von Teilen berücksichtigt

Baugruppen

wenn aktiviert, werden Passungen von Baugruppen berücksichtigt

wenn deaktiviert, werden keine Passungen von Baugruppen berücksichtigt

nur auf erstem Blatt

wenn aktiviert, wird nur auf dem ersten Zeichnungsblatt nach Passungen gesucht

wenn deaktiviert, wird auf allen Blättern nach Passungen gesucht
